Card — the base surface; a hairline plus a quiet lift, never a heavy drop shadow.

```jsx
<Card>Default surface</Card>
<Card tone="ai" accent>Copilot suggestion</Card>
<Card tone="go">QPI 92 — go see them today</Card>
<Card tone="sunk" pad="sm">No research yet — add what you know</Card>
```

Props: `tone` (default/ai/go/sunk), `elevation` (none/sm/md), `pad` (none/sm/md/lg), `accent` (left stripe), `interactive` (hover lift).
