Avatar — a person, company, or foundation. People are circular; orgs render as rounded-squares so the three prospect types stay visually distinct. Falls back to deterministic tinted initials.

```jsx
<Avatar name="Maria Alvarez" size="lg" />
<Avatar name="Hartwell Foundation" kind="org" />
<Avatar name="Dana Reese" ringColor="var(--role-manager)" />
```

Props: `name`, `src`, `size` (sm/md/lg), `kind` (person/org), `ringColor` (ownership ring).
